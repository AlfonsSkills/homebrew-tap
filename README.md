# Homebrew Tap for AlfonsSkills

This repository contains Homebrew formulae for AlfonsSkills projects.

## Installation

```bash
brew tap AlfonsSkills/tap
brew install skillsync
```

Or install directly:

```bash
brew install AlfonsSkills/tap/skillsync
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [skillsync](https://github.com/AlfonsSkills/SkillSync) | Sync skills from Git repositories to 14+ AI coding tools |
